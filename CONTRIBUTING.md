# Contributing to goLyrics

Thanks for contributing to goLyrics.

## Development setup

```bash
npm install
npm run dev
```

## Before opening a PR

- Run `npm run typecheck`
- Run `npm run build`
- Test manually on macOS with Chrome + YouTube

## Commit style

- Keep commits focused and small
- Use clear messages (`fix: ...`, `feat: ...`, `docs: ...`)

## Areas where help is welcome

- Song detection providers (Spotify / Apple Music)
- Better lyric matching and offsets
- Packaging, signing, and updater workflows
- UI polish and accessibility
