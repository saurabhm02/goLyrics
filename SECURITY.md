# Security Policy

## Supported versions

goLyrics is pre-1.0 and under active development. Security fixes are applied to the latest `main` branch.

## Reporting a vulnerability

Please do not open a public issue for sensitive security reports.

Send details to the maintainer directly with:

- Affected version / commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if any)

We aim to acknowledge reports within 72 hours.
